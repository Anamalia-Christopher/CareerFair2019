To run this program, all one needs is a input file as specified in the instruction manual.


In order to avoid needless recursions, the program determines the number of flights for at most 5 flights
since that is even about the realistic number for every individual who wants to fly.


The radius of the earth use in the calculation was 6371km.
The distance is rounded to integer because of significant differences between points on earth


The main program is written in main.py while the program is called in run.py for the sake of simplicity


To use this format, the command format on the command line is :
        python run.py <name-of-input-file>

NOTE : Please ensure that there is no space in the name of the input file or else you are going to get an error
